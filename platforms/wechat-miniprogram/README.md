# 次元绘图微信小程序端

这是次元绘图的微信小程序端，目录可直接导入微信开发者工具。

## 使用

1. 打开微信开发者工具。
2. 导入 `platforms/wechat-miniprogram`。
3. AppID 可先使用测试号或把 `project.config.json` 中的 `appid` 改成自己的小程序 AppID。
4. 首页是 Web 风格的提示词广场，展示提示词和对应示例图。
5. 点击右上角配置按钮，填写 API 格式、API 地址、API Key、模型名和画幅。
6. 回到首页，可输入自定义提示词生成图片；也可点击提示词卡片或“使用”，编辑提示词后再生成。
7. 底部「我的」页可登录（填写头像昵称）、查看累计/收藏/7 天作品统计与生成历史。

> 提示词广场数据源由小程序内置并锁定，用户不可在端内修改。

## 支持

- OpenAI 兼容图片接口：`/images/generations`
- Gemini `generateContent` 图片返回
- base64 图片预览和保存
- 本地保存 API 配置
- Web 风格提示词广场：首页搜索、分类、收藏、详情弹层、示例图列表
- 自定义提示词生图、列表提示词编辑后生图
- 独立配置页：接口、模型、画幅配置（提示词数据源已内置锁定）
- 「我的」个人中心：本地登录（头像昵称）、累计/收藏/7 天作品统计
- 本地生成历史：图片存本机（`USER_DATA_PATH`），满 7 天自动清理

## 注意

微信小程序正式运行时必须在小程序后台配置合法请求域名。开发调试时可在微信开发者工具里勾选“不校验合法域名、web-view、TLS 版本以及 HTTPS 证书”。

若使用 OpenAI 等返回临时图片 URL 的接口，需在小程序后台把图片主机域名加入 **downloadFile 合法域名**，否则正式版无法把历史图片落地保存（开发工具勾选“不校验合法域名”时不受影响）。
